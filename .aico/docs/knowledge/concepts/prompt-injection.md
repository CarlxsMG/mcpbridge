# Prompt Injection

**Path:** `prompt injection`  
**Kind:** `concept`

> A security threat where adversarial content embedded in LLM tool descriptions attempts to override model behavior via imperative phrases like 'ignore previous instructions' or 'act as'. The sanitizer defends against this by pattern-matching and stripping known phrases.
