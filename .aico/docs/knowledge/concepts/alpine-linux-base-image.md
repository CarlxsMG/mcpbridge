# Alpine Linux Base Image

**Path:** `alpine linux base image`  
**Kind:** `concept`

> A minimal Linux distribution (~5 MB) used as the container OS. Reduces image size significantly compared to Debian/Ubuntu-based images but uses musl libc which may affect native addon compatibility.
