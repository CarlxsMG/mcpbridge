# Sec-Fetch-Site Header

**Path:** `sec-fetch-site header`  
**Kind:** `concept`

> A browser-injected request header used here as a heuristic to distinguish browser-initiated requests from server-to-server calls. Its presence without an Origin header triggers a 403.
